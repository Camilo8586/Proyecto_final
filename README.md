# Proyecto_final

Infoanalytics es un programa de python en el que puedes cargar archivos y generar métricas utiles para el analisis de datos, el programa soporta archivos de tipo: .txt, .csv, .docx, .xlxs y .pdf.
Adicionalmente, el programa soporta textos en los idiomas de español, inglés, francés, portugués y alemán.
Cuando el programa genere un reporte, tienes la opción de exportar el reporte como un archivo .txt y simplemente visualizarlo en la consola de python.

Dentro de las carpetas del proyecto tendrás herramientas extra para utilizar el programa de manera más facil y entender su funcionamiento:

- Dentro de la carpeta "data" tendrás varios archivos con los que puedes probar el funcionamiento del programa, esto en caso de que aún no tengas un archivo para utilizar el programa.
- Dentro de la carpeta "docs" dentrás unas diapositivas que explican de mejor manera el proyecto, sus objetivos, y como este fue planteado. Así como 2 archivos .txt en los que puedes ver cómo son los reportes de texto y datos generados por el programa.
- Dentro de la carpeta "main" encontrarás un instructivo para instalar las librerias necesarias para el correcto funcionamiento del programa, es crucial que realices esta instalación
- Finalmente, dentro de la carpeta "src" encontrarás el código de python del programa.

Para ejecutar el programa puedes utilizar el código de "src" y ejecutarlo en la aplicación que desees, o puedes usar el ejecutable (.exe) que se encuentra en los releases del proyecto, este ejecutable es un .zip que contiene todo lo necesario para el funcionamiento del proyecto como un ejecutable.
Si solo te interesa utilizar el programa, puedes solo instalar el .zip del ejecutable en los releases, lo contenido en las carpetas del proyecto solo son herramientas para una mejor compresión del programa.
